baekjoon 2747번 피보나치 수열

기본적인 피보나치 수열 코드 -> 시간 초과

```
def Fibonacci(n):
    if n == 0 or n == 1 :
        return n
    else:
        return Fibonacci(n-1) + Fibonacci(n-2)

N = int(input())
print(Fibonacci(N))

```

재귀를 하면 똑같은 수를 계속 호출에서 시간초과가 난다.

그러므로 피보나치 리스트를 만들어 중복호출을 없앤다.
```
Fibonacci_list = [0,1]


N = int(input())

for i in range(2,N+1):
    Fibonacci_list.append(Fibonacci_list[i-1]+Fibonacci_list[i-2])

print(Fibonacci_list[N])

```

하지만 굉장히 파이써닉한 생각이다. 그래서 리스트 길이를 먼저 정하고 계산했다.

```
Fibonacci_list = [0]*46
Fibonacci_list[1] = 1

N = int(input())

for i in range(2,N+1):
    Fibonacci_list[i] = Fibonacci_list[i-1]+Fibonacci_list[i-2]

print(Fibonacci_list[N])

```
